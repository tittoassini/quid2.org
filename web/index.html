Quid2 is a set of mini-specifications for distributed information systems:

Quid2 Names (coming soon)
Distributed, persistent data dictionary.
Quid2 Typed (coming soon)
Self-describing data.
Quid2 Model
A generic data model.
Quid2 IO (coming soon)
A data-interchange and data-storage binary format.
Quid2 Binary.
Simple and efficient data encoding.
